Raydiant makes it easy to build apps for digital signage and kiosks.

- **Use the tools you love**: Build apps using your favorite web frameworks.
- **Customization**: Add inputs to customize your apps in the Raydiant Dashboard.
- **On-device caching:** Built-in offline support so your apps are always running.
- **Backend APIs:** APIs to help you build delightful apps in record time.
